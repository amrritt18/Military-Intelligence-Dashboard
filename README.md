# 📌 Overview

The **AI-Based Military Intelligence Dashboard** is an end-to-end Machine Learning application developed using **Python**, **Streamlit**, **Scikit-learn**, **Prophet**, and **Plotly** for analyzing historical terrorism incidents from the **Global Terrorism Database (GTD)**.

The project combines:

- Machine Learning
- Time-Series Forecasting
- Interactive Data Visualization
- Geographic Intelligence Mapping
- AI-assisted Intelligence Reporting

into a single interactive dashboard.

Unlike traditional dashboards that only visualize data, this system also predicts attack types, estimates threat levels, forecasts future incidents, and automatically generates intelligence reports to assist strategic decision-making.

---

# 🚀 Project Objectives

The primary objective of this project is to demonstrate how Artificial Intelligence and Data Science techniques can be applied to historical terrorism data for intelligence analysis.

The dashboard provides:

- Historical terrorism analysis
- Country-wise intelligence insights
- Global threat visualization
- Machine learning based attack prediction
- Threat level estimation
- Future attack forecasting
- AI-generated intelligence reports
- Interactive data exploration

# ⚙️ Installation Guide

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Military-Intelligence-Dashboard.git
```

---

## 2️⃣ Move to the Project Directory

```bash
cd Military-Intelligence-Dashboard
```

---

## 3️⃣ Create a Virtual Environment

```bash
python -m venv venv
```

---

## 4️⃣ Activate the Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

---

## 5️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

---

## 6️⃣ Launch the Dashboard

```bash
streamlit run app.py
```

The application will automatically open in your default web browser.
